# 26 — Imperativo

## Imperativo 2ª singular (familiar, "¡tú!")

Toma la **2ª persona presente** del verbo y quita la **-i** final si la tiene. Generalmente queda solo el stem.

| Inf. | 2ª pres. | Imperativo |
|---|---|---|
| runāt | runā | **runā!** |
| strādāt | strādā | **strādā!** |
| lasīt | lasi | **lasi!** *(la -i queda)* |
| rakstīt | raksti | **raksti!** |
| iet | ej | **ej!** |
| nākt | nāc | **nāc!** |
| ēst | ēd | **ēd!** |
| dot | dod | **dod!** |
| pirkt | pērc | **pērc!** |

## Imperativo formal y plural (jūs)

Añade **-iet** al stem:

| Inf. | Imperativo |
|---|---|
| runāt | runājiet! |
| strādāt | strādājiet! |
| lasīt | lasiet! |
| iet | ejiet! |
| nākt | nāciet! |
| ēst | ēdiet! |
| dot | dodiet! |

## Reflexivos

- 2sg: **stem + -ies**: *mācies!, atpūties!, celies!, klausies!*
- jūs: **stem + -ieties**: *mācieties!, atpūtieties!, celieties!, klausieties!*

## Negativo: *ne-* + imperativo

- Nesaki! (¡No digas!)
- Neej! (¡No vayas!)
- Nelasiet! (¡No lean!)
- Nemācies! (¡No estudies!)

## Cortesía: cómo suavizar órdenes

> Las órdenes directas suenan rudas. Para pedir educadamente:

| Forma | Sirve para |
|---|---|
| **Lūdzu, _____ ** | Por favor, _____ |
| **Vai jūs varētu _____ ?** | ¿Podría usted _____? |
| **Vai es drīkstu _____ ?** | ¿Me permite _____? |
| **Es gribētu _____** | Quisiera _____ |
| **Atvainojiet, vai _____ ?** | Disculpe, ¿podría _____? |

Ejemplos:
- *Atver durvis!* (¡Abre la puerta!) — directo
- *Vai jūs varētu atvērt durvis?* (¿Podría abrir la puerta?) — cortés

## Errores comunes

1. **Usar el infinitivo como imperativo**: *Lasīt!* ❌ → *Lasi!* ✅
2. **Olvidar -iet en formal/plural**: *Nāc!* a tu jefe → mal. *Nāciet!* ✅
3. **Olvidar -ies/-ieties en reflexivos**: *Māci!* (= ¡enseña!) ≠ *Mācies!* (= ¡estudia!)
4. **Pegar el imperativo a un infinitivo en negativo**: *Ne ej!* ❌ → *Neej!* ✅
