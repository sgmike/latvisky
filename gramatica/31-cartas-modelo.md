# 31 — Cartas modelo A2

## La fórmula universal

Toda carta A2 sigue la misma estructura:

```
1. SALUDO           (1 línea)
2. INTRODUCCIÓN     (1 frase — motivo)
3. CUERPO           (2-3 frases — desarrollo)
4. CIERRE           (1 frase — pedido o agradecimiento)
5. DESPEDIDA        (1 línea)
6. FIRMA            (tu nombre)
```

Total: **50-80 palabras** (7-9 frases cortas).

## Saludos por registro

| Tipo | Saludo |
|---|---|
| Informal (amigo) | Sveiks, [nombre m]! / Sveika, [nombre f]! |
| Neutro | Labdien, [nombre]! |
| Formal | Cienījamais [Sr.] kungs, / Cienījamā [Sra.] kundze, |

## Despedidas por registro

| Tipo | Despedida |
|---|---|
| Muy informal | Atā! / Sveicinot, |
| Cálido | Ar mīlestību, / Skūpstu, |
| Neutro | Ar labākajiem vēlējumiem, |
| Formal | Ar cieņu, |

## Frases-fórmula reutilizables

### Apertura

- Es ceru, ka tev iet labi. — Espero que te vaya bien.
- Cik sen mēs neesam runājuši! — ¡Cuánto sin hablar!
- Es tev rakstu, jo... — Te escribo porque...
- Es ļoti priecājos saņemt tavu vēstuli. — Me alegré mucho de recibir tu carta.

### Para invitar

- Es gribētu tevi uzaicināt uz... — Quisiera invitarte a...
- Vai tu vari atnākt...? — ¿Puedes venir...?
- Mēs sāksim plkst. ... — Empezaremos a las...

### Para agradecer

- Liels paldies par... — Muchas gracias por...
- Tu esi tik laipnis/laipnā. — Eres tan amable.
- Es ļoti novērtēju... — Aprecio mucho...

### Para disculparse

- Atvainojos, ka... — Disculpa por...
- Ļoti žēl, ka... — Es una pena que...
- Vēlreiz atvainojos. — Disculpas otra vez.

### Para pedir info

- Vai jūs varētu man pateikt...? — ¿Podría decirme...?
- Es interesējos par... — Me interesa...
- Gaidu jūsu atbildi. — Espero su respuesta.

### Cierre cálido

- Gaidu drīzu atbildi! — ¡Espero respuesta pronto!
- Drīz redzēsimies! — ¡Pronto nos vemos!
- Ļoti gaidu! — ¡Lo espero con muchas ganas!

## 3 modelos completos

### A. Invitación

```
Sveiks, Andri!
Es ceru, ka tev iet labi. Es tev rakstu, jo nākamajā
sestdienā svinēšu savu dzimšanas dienu.
Mēs sāksim plkst. 19:00 manā mājā, Brīvības 100.
Vai tu vari atnākt? Lūdzu, atbildi man līdz trešdienai.
Ļoti gaidīšu tevi!
Ar mīlestību,
Pedro
```

### B. Agradecimiento

```
Sveika, Anna!
Liels paldies par to, ka aizvedi mani uz lidostu! Bez
tavas palīdzības es noteikti nokavētu reisu.
Es ļoti novērtēju tavu laipnību. Tu esi īsta draudzene.
Kad atgriezīšos, uzaicināšu tevi uz vakariņām.
Drīz tiksimies!
Skūpstu,
Mārta
```

### C. Disculpa

```
Cienījamais Bērziņa kungs,
Atvainojos, ka vakar nevarēju ierasties uz sapulci.
Es saslimu un visu dienu pavadīju gultā.
Šobrīd jūtos labāk. Vai mēs varētu pārcelt sapulci
uz nākamo nedēļu?
Vēlreiz atvainojos.
Ar cieņu,
Pedro García
```

## Reglas no negociables

1. **Cuenta palabras**: 50-80 ✅. <50 o >80 = puntos restados.
2. **Diacríticos obligatorios** (ā, ē, ī, ū, č, š, ž, ķ, ģ, ļ, ņ).
3. **Al menos 2 conectores** en la carta (jo, bet, tāpēc, ja, un…).
4. **Saludo + despedida coherentes** (no mezcles formal e informal).
5. **Diferentes verbos** — no repitas el mismo 4 veces.
