# 38 — Himno nacional: *Dievs, svētī Latviju!*

> ⚠️ Para el **examen de residencia permanente** debes ser capaz de **recitar el himno**. No cantar — recitar. Lo evalúan oficialmente.

## Letra oficial

```
Dievs, svētī Latviju,
Mūs' dārgo tēviju,
Svētī jel Latviju,
Ak, svētī jel to!

Kur latvju meitas zied,
Kur latvju dēli dzied,
Laid mums tur laimē diet,
Mūs' Latvijā!
```

## Traducción al español

```
Dios, bendice a Letonia,
Nuestra querida patria,
¡Bendice ya a Letonia,
Oh, bendícela ya!

Donde las hijas letonas florecen,
Donde los hijos letones cantan,
Déjanos allí bailar de felicidad,
¡En nuestra Letonia!
```

## Pronunciación con acentos

```
DIEVS, SVĒTĪ LAT-vi-ju,
mūs' DĀR-go TĒ-vi-ju,
SVĒTĪ jel LAT-vi-ju,
ak, SVĒTĪ jel to!

kur LAT-vju MEI-tas zied,
kur LAT-vju DĒ-li dzied,
laid mums tur LAIMĒ diet,
mūs' LAT-vi-jā!
```

**Recordatorio**: el acento letón siempre en la **primera sílaba**. Las vocales largas (ā, ē, ī, ū) son **más largas**, no acentuadas.

## Vocabulario del himno

| Palabra | Significado | Notas |
|---|---|---|
| **Dievs** | Dios | |
| **svētī** | bendice | imperativo de *svētīt* |
| **mūs'** | nuestro/a | contracción poética de *mūsu* |
| **dārgo** | querida | adj. acc fem (concuerda con *tēviju*) |
| **tēvija** | patria | de *tēvs* (padre) |
| **jel** | pues, ya | partícula enfática |
| **ak** | oh | interjección |
| **latvju** | de los letones | gen pl, forma poética (estándar: *latviešu*) |
| **meitas** | hijas, chicas | nom pl de *meita* |
| **zied** | florecen | 3ª pres de *ziedēt* |
| **dēli** | hijos, chicos | nom pl de *dēls* |
| **dzied** | cantan | 3ª pres de *dziedāt* |
| **laid** | deja, permite | imperativo de *laist* |
| **laime** | felicidad | |
| **diet** | bailar | forma poética arcaica de *dejot* |

## Datos del himno

- **Letra y música**: Kārlis Baumanis (1834-1905)
- **Año**: 1873
- **Adoptado**: 7 de junio de 1920
- **Prohibido en**: período soviético (1944-1990)
- **Readoptado**: 15 de febrero de 1990

## Reglas de protocolo

- Se canta o recita **de pie**.
- Se quita el sombrero / gorra.
- Mano derecha sobre el corazón (no obligatorio pero respetuoso).
- Mirando hacia la bandera si la hay.

## Cómo memorizarlo

### Estrategia 1: por bloques de 2 versos

Memoriza primero las 2 primeras líneas, luego las 4 primeras, luego las 6, luego completo. Sin agobio.

### Estrategia 2: rimas

Observa las rimas:
- 1ª estrofa: Latviju / tēviju / Latviju / jel to
- 2ª estrofa: zied / dzied / diet / Latvijā

### Estrategia 3: pronunciación con audio

Busca en YouTube "Dievs, svētī Latviju" — hay versión oficial cantada que te ayuda a aprender entonación y pronunciación.

## Variantes y errores típicos

- ❌ *Latvija* (nom) → ✅ *Latviju* (acc) — porque "bendice **a** Letonia".
- ❌ *mūsu* (forma normal) → en el himno se usa *mūs'* (contracción poética).
- ❌ *dejot* → ✅ *diet* (forma arcaica usada en el himno; no la uses en habla normal).
