# 19 — Mapa completo de preposiciones y casos

## Resumen rápido (memorízalo)

| Preposición | Caso (sg / pl) | Significa |
|---|---|---|
| **uz** | acc / acc | hacia (dirección) |
| **ar** | acc / dat | con |
| **par** | acc / acc | sobre, como |
| **caur** | acc / acc | a través de |
| **pret** | acc / acc | contra, hacia |
| **pa** | acc, dat (depende) | por (a lo largo) |
| **no** | gen / gen | de, desde |
| **bez** | gen / gen | sin |
| **pēc** | gen / gen | después de |
| **pirms** | gen (tradicional) / dat (moderno) | antes de |
| **pie** | gen / gen | en casa de, junto a |
| **aiz** | gen / gen | detrás de |
| **virs** | gen / gen | encima de |
| **zem** | gen / gen | debajo de |
| **dēļ** | gen / gen | a causa de |
| **līdz** | dat / dat | hasta |
| **starp** | dat / dat | entre |
| **blakus** | dat / dat | al lado de |
| **pretī** | dat / dat | enfrente de |

Y el caso **locativo** se usa **SIN preposición**.

## Categorías mnemónicas

### Las "tres uz"

- **uz** = dirección hacia un lugar (uz Rīgu)
- **uz** = sobre la superficie de (uz galda)
- **uz** = para (un objetivo) (uz mājām = a casa)

### Preposiciones de tiempo

- **pēc + gen**: pēc darba (después del trabajo)
- **pirms + gen/dat**: pirms vakara/vakara (antes de la noche)
- **no + gen**: no rīta (desde la mañana)
- **līdz + dat**: līdz pusnaktij (hasta medianoche)

### Preposiciones de lugar (estáticas vs dirección)

| Idea | Estructura |
|---|---|
| En, dentro | **Locativo solo** (mājā) |
| Hacia | **uz + acc** (uz māju) |
| De, desde | **no + gen** (no mājas) |
| Junto a | **pie + gen** (pie mājas) |
| Detrás de | **aiz + gen** (aiz mājas) |
| Encima de (sobre) | **virs + gen** (virs mājas) — o **uz + gen** "encima" |
| Debajo de | **zem + gen** (zem mājas) |
| Al lado de | **blakus + dat** (blakus mājai) |
| Enfrente de | **pretī + dat** (pretī mājai) |

## Casos sin preposición que expresan caso

A veces el **caso solo** ya dice todo:

- **Locativo**: Rīgā (en Riga), maijā (en mayo)
- **Acusativo de duración**: divus gadus (durante dos años)
- **Genitivo posesivo**: mātes māja (la casa de la madre)
- **Dativo "para edad"**: Man ir 30 gadi.

## Errores comunes

1. **Confundir *uz + acc* con locativo**:
   - dirección: *Es eju uz veikalu*
   - ubicación: *Es esmu veikalā*

2. **Usar nominativo tras preposición**:
   - ❌ *bez cukurs* → ✅ *bez cukura*
   - ❌ *ar piens* → ✅ *ar pienu*

3. **Olvidar que *blakus, pretī* van con dativo**:
   - ❌ *blakus banka* → ✅ *blakus bankai*

4. **Calcar "en + acc" del español "ir a"**: *Es eju mājā* ❌ (significa "voy dentro de la casa", no "voy a casa"). ✅ *Es eju mājās* (a casa, lit. "a las casas") o *Es eju uz mājām*.

## Ejercicios

### Ej. — Caso correcto

1. Es eju **uz _____** (skola).
2. Tas ir **pretī _____** (banka).
3. Tēja **bez _____** (cukurs).
4. Mēs strādājam **līdz _____** (vakars).
5. Mēs sēžam **pie _____** (galds).
6. Es runāju **par _____** (Latvija).
7. **Blakus _____** (māja) ir parks.
8. **Pēc _____** (darbs) es eju mājās.
9. Bērns spēlē **zem _____** (galds).
10. Mēs braucam **caur _____** (mežs). *(bosque)*

1. uz **skolu** *(acc)*
2. pretī **bankai** *(dat)*
3. bez **cukura** *(gen)*
4. līdz **vakaram** *(dat)*
5. pie **galda** *(gen)*
6. par **Latviju** *(acc)*
7. blakus **mājai** *(dat)*
8. pēc **darba** *(gen)*
9. zem **galda** *(gen)*
10. caur **mežu** *(acc)*
