# 22 — Futuro (nākotne)

## Concepto

Una única forma de futuro, MUY regular. Se aplica a:
- planes (*Rīt es strādāšu*)
- predicciones (*Rīt līs*)
- promesas (*Es tev piezvanīšu*)

> No existe perífrasis "voy a + inf" en letón. Solo el futuro simple.

## Terminaciones

| Persona | Terminación |
|---|---|
| es | **-šu** |
| tu | **-si** |
| viņš/viņa | **-s** |
| mēs | **-sim** |
| jūs | **-siet** |
| viņi/viņas | **-s** |

Reflexivos: **-šos, -sies, -sies, -simies, -sieties, -sies**.

## Patrón por grupos

### Grupo 2 (-āt, -ot): stem + terminación

| Inf. | 1sg | 3 |
|---|---|---|
| strādāt | strādāšu | strādās |
| runāt | runāšu | runās |
| dzīvot | dzīvošu | dzīvos |

### Grupo 3 (-īt, -ēt): infinitivo sin -t + terminación

| Inf. | 1sg | 3 |
|---|---|---|
| lasīt | lasīšu | lasīs |
| rakstīt | rakstīšu | rakstīs |
| redzēt | redzēšu | redzēs |
| gribēt | gribēšu | gribēs |

### Grupo 1 (monosilábicos): stems específicos

| Inf. | 1sg |
|---|---|
| būt | **būšu** |
| iet | **iešu** |
| nākt | **nākšu** |
| ēst | **ēdīšu** |
| dzert | **dzeršu** |
| dot | **došu** |
| ņemt | **ņemšu** |
| pirkt | **pirkšu** |
| sākt | sākšu |
| varēt | varēšu |
| redzēt | redzēšu |

### Reflexivos

- mācīties → **mācīšos**, mācīsies, mācīsies, mācīsimies, mācīsieties, mācīsies
- atpūsties → atpūtīšos…
- celties → celšos…

## Marcadores temporales del futuro

| Letón | Español |
|---|---|
| rīt | mañana |
| parīt | pasado mañana |
| nākamajā nedēļā / mēnesī / gadā | la próxima semana / mes / año |
| pēc + tiempo en gen/dat | dentro de … |
| drīz | pronto |
| vēlāk | más tarde |
| kādreiz | algún día |

## Errores comunes

1. **Calcar perífrasis "voy a + inf"**: no existe. *Es eju mācīties* ≠ "voy a estudiar".
2. **Stem incorrecto en grupo 1**: *Es ēstšu* ❌ → *Es ēdīšu* ✅.
3. **Olvidar -šos en reflexivos**: *Es mācīšu* ❌ (= grupo 3 sin reflexivo, "enseñaré") → *Es mācīšos* ✅ (estudiaré).

## Ejercicios

### Ej. — Pasa al futuro

1. Es strādāju Rīgā. → ___
2. Mēs lasām grāmatu. → ___
3. Viņa raksta vēstuli. → ___
4. Tu esi mājās? → ___
5. Bērni ēd brokastis. → ___
6. Es atpūšos pludmalē. → ___

<details>
<summary>Respuestas</summary>

1. **Es strādāšu Rīgā.**
2. **Mēs lasīsim grāmatu.**
3. **Viņa rakstīs vēstuli.**
4. **Tu būsi mājās?**
5. **Bērni ēdīs brokastis.**
6. **Es atpūtīšos pludmalē.**

</details>
