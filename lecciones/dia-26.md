## ⏱ Bloque 2 — El imperativo (10 min)

Lee `gramatica/26-imperativo.md`. Lo esencial:

### Imperativo familiar (2ª singular — *tú*)

Toma la **2ª persona presente** del verbo y QUITA la **-i** final (si la tiene). Generalmente queda solo el stem.

| Inf. | 2ª pres. (tu) | **Imperativo (¡tú!)** |
|---|---|---|
| runāt | runā | **runā!** |
| strādāt | strādā | **strādā!** |
| lasīt | lasi | **lasi!** |
| rakstīt | raksti | **raksti!** |
| iet | ej | **ej!** |
| nākt | nāc | **nāc!** |
| ēst | ēd | **ēd!** |
| dot | dod | **dod!** |
| dzert | dzer | **dzer!** |
| pirkt | pērc | **pērc!** |

### Imperativo formal y plural (jūs)

Añade **-iet** al stem (o **-jiet** si el stem termina en vocal):

| Inf. | **Imperativo (jūs/ustedes)** |
|---|---|
| runāt | **runājiet!** |
| strādāt | **strādājiet!** |
| lasīt | **lasiet!** |
| iet | **ejiet!** |
| nākt | **nāciet!** |
| ēst | **ēdiet!** |
| dot | **dodiet!** |
| pirkt | **pērciet!** |

### Reflexivos

- 2sg: stem + **-ies**: *mācies!* (¡estudia!), *atpūties!* (¡descansa!), *celies!* (¡levántate!), *klausies!*
- jūs: stem + **-ieties**: *mācieties!*, *atpūtieties!*, *celieties!*, *klausieties!*

### Negativo: *ne-* + imperativo

- **Nesaki!** (¡No digas!)
- **Neej!** (¡No vayas!)
- **Nelasiet!** (¡No lean!)
- **Nemācies!** (¡No estudies!)

### Errores comunes

1. **Usar el infinitivo como orden**: *Lasīt!* ❌ → *Lasi!* ✅
2. **Olvidar -iet en plural/formal**: *Nāc!* a un grupo o a tu jefe → ❌. *Nāciet!* ✅
3. **Olvidar -ies/-ieties en reflexivos**: *Māci!* (significa "¡enseña!", no "estudia") → *Mācies!* ✅

## ⏱ Bloque 3 — Frases sociales: cortesía (5 min)

### Para suavizar peticiones (no parecer ordinario)

| Letón | Español |
|---|---|
| **Lūdzu, _____** | Por favor, _____ |
| **Vai jūs varētu _____ ?** | ¿Podría usted _____? |
| **Vai es drīkstu _____ ?** | ¿Puedo (me permite) _____? |
| **Es gribētu _____** | Quisiera _____ *(condicional cortés)* |
| **Atvainojiet, _____** | Disculpe, _____ |
| **Vai jums būtu _____ ?** | ¿Tendría usted _____? *(condicional)* |

### Frases de saludo/despedida formal

- **Labdien!** — Buenos días/tardes (formal).
- **Sveicināti!** — ¡Saludos! (formal).
- **Uz redzēšanos!** — Hasta luego (formal).
- **Visu labu!** — ¡Que le vaya bien!
- **Veiksmi!** — ¡Suerte!
- **Priecājos jūs satikt!** — Encantado de conocerle.
- **Liels paldies par jūsu palīdzību.** — Muchas gracias por su ayuda.

### Frases para sobrevivir en un restaurante

- **Vai ir brīvs galds?** — ¿Hay mesa libre?
- **Vai es drīkstu rezervēt galdu uz divām personām?** — ¿Puedo reservar mesa para dos personas?
- **Lūdzu, ēdienkarti.** — La carta, por favor.
- **Es gribētu _____.** — Quisiera _____.
- **Kāds ir dienas piedāvājums?** — ¿Cuál es el menú del día?
- **Vai šeit ir veģetāras opcijas?** — ¿Hay opciones vegetarianas?
- **Ļoti garšīgi, paldies!** — ¡Muy delicioso, gracias!
- **Rēķinu, lūdzu!** — ¡La cuenta, por favor!

### Frases para el hotel

- **Man ir rezervācija uz vārda _____.** — Tengo una reserva a nombre de _____.
- **Es vēlos rezervēt istabu uz trim naktīm.** — Quisiera reservar habitación tres noches.
- **Cikos sākas brokastis?** — ¿A qué hora empieza el desayuno?
- **Vai istabā ir Wi-Fi?** — ¿Hay Wi-Fi en la habitación?
- **Es vēlos izrakstīties.** — Quisiera hacer check-out.

### Ejercicio 2 — Imperativo formal/plural (jūs)

1. lasīt → ___
2. nākt → ___
3. dot → ___
4. atpūsties → ___

### Ejercicio 3 — Suavizar órdenes

Pasa estas órdenes a forma cortés:

1. Dod man kafiju! → Vai jūs varētu ___?
2. Saki man! → Es gribētu ___ jautāt.
3. Pasūtīt! → Es gribētu pasūtīt ___.

### Ejercicio 4 — Diálogo en el restaurante

Traduce:

1. — Buenos días, ¿hay mesa libre para tres personas?
2. — Sí, por favor pase. ¿Qué desean beber?
3. — Yo quisiera agua, mi esposa un café.
4. — ¿Tienen opciones vegetarianas?
5. — La cuenta, por favor. ¿Aceptan tarjeta?
6. — Sí, claro. Aquí tiene su recibo. ¡Que les vaya bien!
