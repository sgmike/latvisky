# Día 6 — Números, hora, días, meses, fechas

**Objetivo del día**: contar hasta 1000, decir la hora, manejar días de la semana, meses y dar fechas completas. Imprescindible para tiendas, transporte y trámites.

---

## ⏱ Bloque 1 — Repaso (2 min)

Sin mirar arriba, conjuga *iet* y *ēst* en yo, tú, nosotros.

## ⏱ Bloque 2 — Números (10 min)

Lee `gramatica/07-numeros-hora-fechas.md`. Lo crítico:

### Del 0 al 20

| | | | |
|---|---|---|---|
| 0 nulle | 6 seši | 12 divpadsmit | 18 astoņpadsmit |
| 1 viens / viena | 7 septiņi | 13 trīspadsmit | 19 deviņpadsmit |
| 2 divi / divas | 8 astoņi | 14 četrpadsmit | 20 divdesmit |
| 3 trīs | 9 deviņi | 15 piecpadsmit | |
| 4 četri / četras | 10 desmit | 16 sešpadsmit | |
| 5 pieci / piecas | 11 vienpadsmit | 17 septiņpadsmit | |

> ⚠️ **Concordancia de género**: *viens, divi, četri, pieci…* tienen forma femenina *viena, divas, četras, piecas*. Úsala según el sustantivo.
> - **viens vīrs** (un hombre) / **viena māsa** (una hermana)
> - **divi brāļi** (dos hermanos) / **divas māsas** (dos hermanas)
> Desde 11 (vienpadsmit) en adelante **no cambia** por género.

### Decenas y combinaciones

- 20 **divdesmit** · 21 **divdesmit viens** · 25 **divdesmit pieci**
- 30 **trīsdesmit** · 40 **četrdesmit** · 50 **piecdesmit**
- 60 **sešdesmit** · 70 **septiņdesmit** · 80 **astoņdesmit** · 90 **deviņdesmit**
- 100 **simts** (también *viens simts*)
- 200 **divi simti** · 300 **trīs simti** · 500 **pieci simti**
- 1000 **tūkstotis** · 2000 **divi tūkstoši**
- 1.000.000 **miljons**

### Truco para escribir cualquier número

Letón pega las decenas como una sola palabra:
- **246** = *divi simti četrdesmit seši*
- **1987** = *tūkstotis deviņi simti astoņdesmit septiņi*

## ⏱ Bloque 3 — La hora (8 min)

### Preguntar y responder

- **Cik ir pulkstenis?** — ¿Qué hora es?
- **Pulkstenis ir + número** — Es la / Son las…
- **Cikos?** — ¿A qué hora?

### Ejemplos

| Hora | Letón |
|---|---|
| 8:00 | Pulkstenis ir **astoņi**. |
| 13:00 | Pulkstenis ir **viens** (pēcpusdienā). |
| 17:30 | Pulkstenis ir **pusseši** *(literal: "media-seis", o sea media para las 6 = 5:30)* |
| 7:15 | Pulkstenis ir **ceturksnis pāri septiņiem** *(cuarto pasado las 7)* |
| 7:45 | Pulkstenis ir **bez ceturkšņa astoņi** *(sin cuarto las 8 = falta un cuarto para las 8)* |

> 💡 **Truco importante**: la palabra **pusseši (5:30)** se forma con *pus-* + el número de la hora que sigue. *Pussešos* = a las 5:30. Para hispanohablantes es contraintuitivo: *pusseši* NO es las 6:30, es las 5:30. Imagina "a medio camino hacia las seis".

### Partes del día

- **no rīta** (por la mañana, 6-12)
- **pēcpusdienā** (por la tarde, 12-18)
- **vakarā** (por la noche, 18-22)
- **naktī** (de madrugada, 22-6)

## ⏱ Bloque 4 — Días, meses y fechas (5 min)

### Días de la semana

| Letón | Español |
|---|---|
| **pirmdiena** | lunes |
| **otrdiena** | martes |
| **trešdiena** | miércoles |
| **ceturtdiena** | jueves |
| **piektdiena** | viernes |
| **sestdiena** | sábado |
| **svētdiena** | domingo |

> 💡 **Memo**: *pirm-* (primer), *otr-* (segundo), *treš-* (tercero), *ceturt-* (cuarto), *piekt-* (quinto), *sest-* (sexto), *svēt-* (sagrado/santo, por domingo).

### Para decir "el lunes" (sin preposición)

- *pirmdien* → forma adverbial. *Es nestrādāju **pirmdien**.* (No trabajo el lunes.)
- *pirmdienās* → los lunes (recurrente).

### Meses

| Letón | Español |
|---|---|
| **janvāris** | enero |
| **februāris** | febrero |
| **marts** | marzo |
| **aprīlis** | abril |
| **maijs** | mayo |
| **jūnijs** | junio |
| **jūlijs** | julio |
| **augusts** | agosto |
| **septembris** | septiembre |
| **oktobris** | octubre |
| **novembris** | noviembre |
| **decembris** | diciembre |

### Fechas

Formato: **día (ordinal locativo) + mes (genitivo) + año (genitivo)**

- **16. maijā** = el 16 de mayo (*16. = sešpadsmitajā*)
- **3. janvārī** = el 3 de enero
- **2026. gadā** = en el año 2026 (lit. "en el año 2026")

> 🔥 Para A2 con que digas **"16. maijā"** y **"2026. gadā"** ya estás bien. La forma ordinal completa (*sešpadsmitajā*) suena culta pero el número en cifras funciona.
