## 🎵 Bloque 1 — *Dievs, svētī Latviju!* (15 min)

> ⚠️ Para el examen de **residencia permanente**, debes ser capaz de **recitar el himno**. No cantar, recitar. Lo evalúan.

### Letra completa (memorízala)

```
Dievs, svētī Latviju,
Mūs' dārgo tēviju,
Svētī jel Latviju,
Ak, svētī jel to!

Kur latvju meitas zied,
Kur latvju dēli dzied,
Laid mums tur laimē diet,
Mūs' Latvijā!
```

### Traducción literal al español

```
Dios, bendice a Letonia,
Nuestra querida patria,
¡Bendice ya a Letonia,
Oh, bendícela ya!

Donde las hijas letonas florecen,
Donde los hijos letones cantan,
Déjanos allí bailar de felicidad,
¡En nuestra Letonia!
```

### Pronunciación clave

Léelo en voz alta con estos acentos:

```
DIEVS, SVĒTĪ LAT-vi-ju,
mūs' DĀR-go TĒ-vi-ju,
SVĒTĪ jel LAT-vi-ju,
ak, SVĒTĪ jel to!

kur LAT-vju MEI-tas zied,
kur LAT-vju DĒ-li dzied,
laid mums tur LAIMĒ diet,
mūs' LAT-vi-jā!
```

### Vocabulario del himno

| Letón | Español |
|---|---|
| Dievs | Dios |
| svētī | bendice *(imperativo de svētīt)* |
| mūs' | nuestro/a *(forma corta de "mūsu")* |
| dārgo | querida *(adj. fem acc)* |
| tēvija | patria |
| jel | pues, ya *(partícula enfática)* |
| ak | oh *(interjección)* |
| latvju | de los letones *(gen pl, forma poética)* |
| meitas | hijas / chicas |
| zied | florecen |
| dēli | hijos / chicos |
| dzied | cantan |
| laid | deja, permite *(imperativo de laist)* |
| laime | felicidad |
| diet | bailar *(forma poética de dejot)* |

### Compositor

- **Letra y música**: **Kārlis Baumanis** (1834-1905)
- Compuesto en **1873** durante el "Despertar Nacional" letón
- Adoptado como himno oficial el **7 de junio de 1920**
- En el período soviético estuvo prohibido; readoptado el **15 de febrero de 1990**

## 🏛 Bloque 2 — Repaso de historia (15 min)
