## ⏱ Bloque 2 — El futuro (nākotne) (15 min)

Lee `gramatica/22-futuro-nakotne.md`. La buena noticia:

> ✨ **El futuro letón es uno de los tiempos más regulares y predecibles.**

### Terminaciones del futuro (verbos no reflexivos)

| Persona | Terminación |
|---|---|
| es | **-šu** |
| tu | **-si** |
| viņš / viņa | **-s** |
| mēs | **-sim** |
| jūs | **-siet** *(coloquial: -sit)* |
| viņi / viņas | **-s** |

Se añaden al **stem del infinitivo** (sin -t). Por grupos:

### Grupo 2 — *strādāt* (trabajar)

| Pron. | Futuro |
|---|---|
| es | **strādāšu** |
| tu | strādāsi |
| viņš | strādās |
| mēs | strādāsim |
| jūs | strādāsiet |
| viņi | strādās |

### Grupo 3 — *lasīt* (leer)

| Pron. | Futuro |
|---|---|
| es | **lasīšu** |
| tu | lasīsi |
| viņš | lasīs |
| mēs | lasīsim |
| jūs | lasīsiet |
| viņi | lasīs |

### Grupo 1 — los irregulares

Los del grupo 1 tienen **stems específicos** para el futuro. Memoriza:

| Inf. | Futuro 1sg | 3 |
|---|---|---|
| **būt** | būšu | būs |
| **iet** | iešu | ies |
| **nākt** | nākšu | nāks |
| **ēst** | **ēdīšu** | ēdīs |
| **dzert** | **dzeršu** | dzers |
| **dot** | **došu** | dos |
| **ņemt** | **ņemšu** | ņems |
| **pirkt** | **pirkšu** | pirks |
| **redzēt** | redzēšu | redzēs |
| **varēt** | varēšu | varēs |
| **gribēt** | gribēšu | gribēs |
| **sākt** | sākšu | sāks |

### Reflexivos en futuro

Las terminaciones son **-šos, -sies, -sies, -simies, -sieties, -sies**:

- mācīties → **mācīšos**, mācīsies, mācīsies, mācīsimies, mācīsieties, mācīsies
- atpūsties → atpūtīšos, atpūtīsies, atpūtīsies, atpūtīsimies, atpūtīsieties, atpūtīsies

### Cuándo se usa el futuro

#### 1. Planes y promesas

- **Rīt es strādāšu** astoņas stundas.
- **Nākamajā vasarā** mēs ceļosim uz Latviju.
- **Es tev piezvanīšu** šovakar.

#### 2. Predicciones

- **Rīt līs.** (Mañana lloverá.)
- **Tu būsi laimīgs.** (Serás feliz.)

#### 3. Acciones probables

- Es **piedalīšos** sapulcē. (Participaré en la reunión.)

### Marcadores temporales del futuro

| Letón | Español |
|---|---|
| **rīt** | mañana |
| **parīt** | pasado mañana |
| **nākamajā nedēļā** | la próxima semana |
| **nākamajā mēnesī** | el próximo mes |
| **nākamajā gadā** | el próximo año |
| **pēc _____** | dentro de _____ *(+ gen)* |
| **drīz** | pronto |
| **vēlāk** | más tarde |
| **kādreiz** | algún día |

### Errores comunes

1. **Calcar la perífrasis "voy a + inf" del español**: no existe en letón. Se usa el futuro directo.
   - *Voy a estudiar* = ❌ *Es eju mācīties* → ✅ *Es mācīšos*.
2. **Olvidar el cambio de stem en grupo 1**: *Es ēstšu* ❌ → *Es ēdīšu* ✅.
3. **Confundir -siet (futuro 2pl) con -sit (coloquial pl)**: ambos válidos, -siet es la forma estándar de examen.

## ⏱ Bloque 4 — Ejercicios (10 min)

### Ejercicio 1 — Pon en futuro

1. Es strādāju Rīgā. → ___
2. Mēs lasām grāmatu. → ___
3. Viņa raksta vēstuli. → ___
4. Tu esi mājās? → ___
5. Mēs ejam kino. → ___
6. Bērni ēd brokastis. → ___
7. Es atpūšos pludmalē. → ___

### Ejercicio 2 — Tus planes

Escribe 6-8 frases sobre tus planes:

- Mañana
- La próxima semana
- En verano / vacaciones largas
- En 2 años
- Algún día (kādreiz)

Usa al menos 5 verbos diferentes en futuro.

### Ejercicio 3 — Traduce

1. Mañana iré al médico.
2. El próximo verano viajaremos a Letonia.
3. ¿A qué hora vendrás esta noche?
4. Dentro de 3 años hablaré letón perfectamente. *(perfectamente = perfekti)*
5. Voy a estudiar (= estudiaré) toda la noche.
