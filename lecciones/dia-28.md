## ⏱ Bloque 2 — Resumen ultra-condensado (10 min)

### Futuro

```
Es ___šu    Mēs ___sim
Tu ___si    Jūs ___siet
Viņš ___s   Viņi ___s

Refl: -šos, -sies, -sies, -simies, -sieties, -sies
```

### Construcción *jā-* (¡memorízala!)

> **[DAT persona] + jā[verbo 3ª pres] + [NOM objeto]**

- *Man jāstrādā.*
- *Man jāpērk maize.* (¡maize en nominativo!)
- *Tev jāmācās latviešu valoda.*

### Comparativos

```
Adj + -āks (m) / -āka (f)        =  más X
vis + -ākais (m) / -ākā (f)     =  el más X
"X más que Y" = X ir [adj-āks] par Y (acc)
```

### Conectores top

- **un** (y), **bet** (pero), **vai** (o)
- **jo** (porque), **tāpēc** (por eso)
- **ja** (si — condicional), **kad** (cuando — temporal)
- **kamēr** (mientras), **pirms** (antes), **pēc tam, kad** (después de)

### Imperativo

```
2sg familiar: stem del 2ª pres (sin -i): runā!, lasi!, ej!, ēd!
2pl/formal:   stem + -iet: runājiet!, lasiet!, ejiet!, ēdiet!
Refl:         -ies (sg), -ieties (pl): mācies!, mācieties!
Negativo:     ne- + imperativo: nelasi!, nesmēķējiet!
```
